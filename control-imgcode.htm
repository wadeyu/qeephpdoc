图像验证码控件

作者：dualface

  • 分类：用户界面控件
  • 最后更新：2009-01-31 17:59
  • 下载次数：1013
  • 项目网址：/projects/control-imgcode

这个图像验证码控件默认状态只显示一个文本输入框，当文本输入框获得焦点后才会显示出验证码。由于每次显示验证码都会发起一个新的请求，所以这种获取焦点后才显示的方式可以避免许多无谓的请求。

使用说明

将 imgcode.php 放到应用程序的 app/control 目录，然后在视图中要显示验证码的地方加入代码：

<?php $this->_control('imgcode');

最后为 default 控制器添加一个 actionImgcode() 方法，内容如下：

function actionImgcode()
{
    return Helper_Imgcode::create();
}

定制

如果希望指定显示验证码的控制器动作，可以在创建 imgcode 控件时传递 udi 参数：

<?php $this->_control('imgcode', 'imgcode', array(
    'udi' => 'common/imgcode'
)); ?>

此时将使用 common 控制器的 imgcode 动作来显示验证码。

除了指定控制器动作，还可以在控制器动作中修改验证码的样式，例如：

function actionImgcode()
{
    // 验证码长度 6 个字符，有效期 900 秒，ttf 模式
    return Helper_Imgcode::create(6, 900, 'ttf');
}

有关 Helper_Imgcode 的详细使用，请参考： http://qeephp.com/…lper-imgcode

在表单中使用

要在表单中使用这个控件，将表单元素的 _ui 属性设置为 imgcode 即可。
