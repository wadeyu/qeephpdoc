xser-php-framework-0.11

作者：vb2005xu

  • 分类：框架
  • 最后更新：2010-04-20 09:28
  • 下载次数：687
  • 项目网址：/projects/xser-php-framework-0_11

// xser php framework ⇒ a simple php5 framework // 1. a dispatcher module // 2. a view module // 3. a database object relations mapping module

开发者: 色色 vb2005xu@qq.com vb2005×u.javaeye.com

版权: 保留修改权,免费使用

v0.11正式版 于 2010年4月19日16:00:00 发布

特色:

  1. MVC 支持
  2. 控制器多级模块,命名空间支持
  3. 使用udi定位视图模板的可继承的模板引擎
  4. 基于文本文件的日志记录组件
  5. 简单的firebug的php服务器端组件
  6. 简单的数据库抽象组件,提供单表操作的完美支持,提供无限分类表实例
  7. 国际化多语言组件,支持多级层次的资源
  8. 功能强大的缓存功能,详见我的博客
  9. 简单的URL路由组件,支持自定义的正向路由解析
  10. 验证码辅助工具,视图辅助方法,自主实现的__init_xu JS框架

欢迎交流…
