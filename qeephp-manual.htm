QeePHP 应用开发权威指南

快速入门教程帮助开发者了解 QeePHP 的开发模式和基本步骤，而权威指南则是开发 QeePHP 应用程序时的全面参考。QeePHP 所有功能的使用方法、使用时机，以及该功能的使用技巧等内容，都会在权威指南中详细阐述。

权威指南按照不同的主题分为了数十篇文章。每篇文章围绕各自的中心编写内容，读者可以直接选择有针对性的文章查看。

目录

1. QeePHP 介绍

  • 基本: 为什么选择 QeePHP 主要特征 QeePHP 的历史 致谢

2. MVC 模式

  • 基本: MVC 工作流程 入口脚本
  • 配置: 启动文件 应用程序设置
  • 执行: 应用程序对象 控制器 上下文对象 助手
  • 视图: 传递数据到视图 视图的继承 视图片段 用户界面控件
  • 其他: URL 路由 访问控制

3. 模型与 ORM

  • 基本: 基础 定义模型 存储模型的数据表 管理模型的属性
  • 安全: 自动验证 保护模型
  • CRUD 操作: 查询 Read 创建 Create 更新 Update 删除 Delete
  • 关联: 对相关联是什么 定义关联 HAS ONE 关联 HAS MANY 关联 BELONGS TO 关联 MANY TO MANY 关联 关联查询
  • 高级: 使用行为插件 元对象
  • 扩展: 事件回调 创建行为插件

4. 表单

  • 基本: 概述 创建表单 操作表单元素
  • 使用: 显示表单 处理数据 不同状态的表单
  • 高级: 动态元素 群组

5. 数据库

  • 基本: 管理数据库连接 使用 schema
  • 查询: 查询方法参考 高级查询
  • 操作: 插入 更新 删除 数据库表达式 数据库事务 命令和存储过程
  • 高级: 多数据库连接 读写分离 分表分库 使用数据库视图
